This weather AI model is predicting future weather based off the weather in the past
to run:
1. Go to requirements.txt to find the requried dependencies
2. Download the dataset from kaggle
    a. pip install kaggle
    a. kaggle datasets download abdelrahman16/weather-seattle -p (filename)/ --unzip
3. Run the train.py changing the path for the dataset.
4. Run RM_Seattle.py (also changing the path of the dataset.)
